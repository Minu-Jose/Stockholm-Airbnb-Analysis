# Stockholm-Airbnb-Analysis
Data analysis of Stockholm Airbnb listings using Python, Pandas and data visualization techniques

## Project Overview
Analysis of Airbnb listings in Stockholm to understand pricing patterns.

## Dataset
Inside Airbnb dataset.

## Questions Explored
- What is the average price?
- Which neighborhoods are expensive?
- Which room types cost more?
- How do reviews affect price?
